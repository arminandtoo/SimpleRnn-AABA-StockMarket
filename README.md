# SimpleRnn-AABA-StockMarket
پیش‌بینی قیمت باز شدن (Open) سهام شرکت AABA  با استفاده از یک شبکه عصبی بازگشتی ساده (SimpleRNN)
# توضیحات پروژه:

هدف این مدل، یادگیری الگوهای موجود در داده‌های تاریخی قیمت سهام برای پیش‌بینی قیمت در روز آینده است. این یک مسئله رگرسیون در تحلیل سری‌های زمانی محسوب می‌شود.
فرآیند کار و معماری مدل

## بارگذاری و آماده‌سازی داده:

داده‌ها از فایل CSV بارگذاری شده و ستون Open استخراج می‌شود.

داده‌ها به دو بخش آموزشی (2500 رکورد اول) و آزمایشی (باقی رکوردها) تقسیم می‌شوند.

## ایجاد دنباله‌ها (Sequences):

مدل بر اساس داده‌های 90 روز گذشته، قیمت روز 91ام را پیش‌بینی می‌کند.

داده‌های آموزشی و آزمایشی بر اساس این پنجره زمانی 90 روزه بازسازی می‌شوند.

## ساختار مدل (SimpleRNN):

مدل به صورت Sequential با استفاده از Keras ساخته شده است.

لایه اول: یک لایه SimpleRNN با 64 واحد (units) و تابع فعال‌سازی tanh.

لایه خروجی: یک لایه Dense با 1 واحد خروجی برای پیش‌بینی مقدار قیمت.

## آموزش مدل:

مدل با بهینه‌ساز (Optimizer) adam و تابع زیان mse (میانگین مربعات خطا) کامپایل شده است.

مدل بر روی داده‌های آموزشی برای 100 ایپاک (Epoch) آموزش داده می‌شود.
داده‌های استفاده شده شامل قیمت روزانه سهام AABA از تاریخ 2006-01-01 تا 2018-01-01 است.
## دیتاست
برای سادگی، تنها ستون Open (قیمت باز شدن سهم) به عنوان ویژگی ورودی مدل انتخاب شده است.
